Overview

This model manage the Profile object. A Profile is a job offer, each Profile has specific partecipation prerequisites and conditions. Each Profile is referenced to an Contest through an Area

Relations

belongs to Area has many Section has many Question through Section has many Request

Validates

  • presence of Area

  • presence of #title

  • presence of #careers_requested

  • numericality of #careers_requested

class Profile < ApplicationRecord
  belongs_to :area, counter_cache: true
  has_many :sections, dependent: :destroy
  has_many :questions, through: :sections
  has_rich_text :body

  accepts_nested_attributes_for :sections

  validates :area, presence: true
  validates :title, presence: true
  validates :careers_requested, presence: true, numericality: { only_integer: true }

  delegate :active?, :ended?, to: :area, allow_nil: true
end
